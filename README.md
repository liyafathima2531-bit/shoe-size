### Project Description

fit finder lets customers quickly check shoe size availability and price by scanning the a qr code scanner.select a brand,enter your size,and instant know if it is in stock it saves time for both shoppers and shopkeepers

### The Problem statement

customers waste time seareching for shoes of a specific size and brand in stores

### The Solution

fit finder solves this by letting customers instantly check size availability and price using qr code

---

## Technical Details

this project is frontend-only web application built using HTML5,css3 and javascript.the structure of the project includes multiple pages:
brand.html-The brand selection page where users choose a shoe brand,size.html-the page where users input their shoe size and check availability.a finish button appears if the size is unavailable,display.html-shows available shoes for the selected brand and size in cards, greeting.html- A friendly thank-you page for the user with an option to return to brand selection.

### Technologies/Components Used 

HTML,CSS and javascript to create brand selection,size check,with finish button and greeting page.
